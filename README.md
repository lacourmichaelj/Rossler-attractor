# Rössler Attractor
Chaotic behavior is best described as small changes in the initial conditions of a system
creating dramatically different outcomes within a system over time. Strange attractors model
chaotic behavior in complex real-world physical phenomena, such as weather or chemical
reactions. The Rössler attractor is defined as a strange attractor. Strange attractors are unique
because although they demonstrate chaotic behavior, they also contain fixed points or
“attractors” that set a pathway for the system. Dr. Otto E. Rössler discovered the strange attractor
that bears his name in 1976. The Rössler attractor is generated by the following system of
equations:
dx/dt = - ( y + z )
dy/dt = x + A y
dz/dt = B + x z – Cz
where a,b,c are constant parameters
